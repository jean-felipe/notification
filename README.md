# Notification App
======

* Clone the project, cd the project and run bundle follow by rake db:setup && rails db:migrate

* Then you can signup and start to create notifitions.

The project tecnologies:
====

* Rails 5
* Action Cable
* Angular
* Bootstrap
* Rspec
* Pusher
* W3schools CSS for cards
